# CSV Cleaner Lite

A beginner-friendly Streamlit app to upload, preview, and clean CSV files in just one click.

## Features

- Upload any `.csv` file
- Remove duplicate rows
- Drop rows with missing values
- Normalize column headers (e.g., `First Name` â†’ `first_name`)
